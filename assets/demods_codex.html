# FFT Implementation

## Overview
Fast Fourier Transform implementation using Cooley-Tukey algorithm.

## Parameters
- **Size**: Must be power of 2 (512, 1024, 2048...)
- **Window**: Hann, Hamming, or Blackman
- **Overlap**: 0-75% (50% recommended)

## Usage
```cpp
FFT fft(1024);
fft.setWindow(HANN);
fft.process(input, output);
```

## Performance
- Complexity: O(n log n)
- Latency: ~0.5ms @ 48kHz
- Memory: 2N complex samples
